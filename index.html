Acerca de mí

Desarrollador Full-Stack con experiencia en la creación de soluciones web modernas y escalables. Trabajo con tecnologías como Node.js, React.js y bases de datos SQL/NoSQL para desarrollar aplicaciones robustas y eficientes. Mi experiencia incluye el diseño y la implementación de APIs, sistemas CRUD, autenticación segura e integraciones que optimizan la experiencia del usuario.

Durante dos años, trabajé en el mantenimiento y optimización de un proyecto empresarial alojado en SAP Business Technology Platform (SAP BTP). Mi experiencia incluye el trabajo con aplicaciones SAP UI5 en NEO como frontend, Node.js con Express y XSJS como backend en CloudFoundry, así como una base de datos SAP HANA Cloud. También participé en la administración de un servidor Java para la ejecución de jobs programados y en la gestión de servicios de almacenamiento de documentos en NEO. Las aplicaciones eran accesibles para los usuarios a través de SAP Cloud Portal Service. Durante este tiempo, me dediqué a aprender y ejecutar tareas clave para asegurar el rendimiento y la estabilidad del proyecto.

Mi enfoque es resolver desafíos complejos y aportar mejoras continuas, asegurando que cada aplicación sea estable, segura y capaz de crecer a la par con las necesidades del negocio.
